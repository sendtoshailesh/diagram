# Visualization Examples

This project demonstrates different ways to generate visuals programmatically.

## Examples Included

1. **Mermaid** - Text-based diagrams
2. **Python Diagrams** - Cloud architecture
3. **D3.js** - Interactive charts
4. **SVG** - Direct vector graphics
5. **Python Matplotlib** - Data visualization
6. **HTML Canvas** - Dynamic graphics
